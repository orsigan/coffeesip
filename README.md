# coffeesip
Shows coffee ascii animations in the terminal

# Build

- you need a C++ compiler
- here, we will use the Gnu G++


## üêß Linux

### Ubuntu / Debian / Linux Mint

```bash
sudo apt update
sudo apt install g++
```

### Fedora

```bash
sudo dnf install gcc-c++
```

### CentOS / RHEL

```bash
sudo dnf groupinstall "Development Tools"
sudo dnf install gcc-c++
```

### Arch Linux / Manjaro

```bash
sudo pacman -S gcc
```
